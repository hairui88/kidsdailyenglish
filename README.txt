
# KidsDailyEnglish — Day 1 (15 words)

这是给孩子用的“点读网页”。包含：index.html、图标、manifest。
支持 iPhone / iPad 加到主屏幕，像 App 一样用。

## 方案一：iCloud Drive（最简单）
1. 在 iPhone/iPad 上打开“文件”App → iCloud Drive。
2. 新建文件夹 `KidsDailyEnglish`。
3. 把这三个文件（index.html, icon-180.png, manifest.webmanifest）**原样**放进该文件夹。
4. 在电脑或 iPad 上打开 https://www.icloud.com/iclouddrive ，找到 `index.html`。
5. 右键“共享”→“复制链接”→ 权限改为“任何知道链接的人”。
6. 把链接发给孩子。孩子用 Safari 打开 → 分享 → 添加到主屏幕。

> 注意：某些情况下 iCloud 直链会变成下载。若无法在线打开，请使用方案二。

## 方案二：GitHub Pages（最稳）
1. 访问 https://github.com 并登录（无账号先注册）。
2. 新建仓库：名字填 `kidsdailyenglish`，公开（Public）。
3. 上传本文件夹内三个文件（index.html, icon-180.png, manifest.webmanifest）。
4. 打开仓库 Settings → Pages → Source 选 `Deploy from a branch`，Branch 选 `main /root`，点 Save。
5. 1 分钟后会出现一个网址，例如：`https://<你的用户名>.github.io/kidsdailyenglish/`。
6. 孩子用 Safari 打开这个网址 → 分享 → 添加到主屏幕。

## 使用方法
- 🔊 “Word”按钮：读单词
- 🔊 “Sentence”：读例句
- 👁 Hide/Show：隐藏中文测记忆
- ➡️ Next：下一词
- 🔀 Shuffle：打乱顺序

祝学习愉快！
